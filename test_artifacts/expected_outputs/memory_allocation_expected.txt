## Phase 4: Free All

### After freeing all blocks
Expected:
  used_memory: 0
  free_memory: 2048
  num_free_blocks: 1  # All coalesced
  largest_free_block: 2048
  fragmentation: 0.0  # No fragmentation

## Correctness Criteria
✓ used_memory + free_memory = 2048 (always)
✓ All allocation_ids are unique and ≥ 0
✓ largest_free_block ≤ free_memory
✓ fragmentation in range [0.0, 1.0]
✓ Successful allocations when free_memory ≥ requested_size
